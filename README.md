This is a streamlit application in which there are 2 pages . First page shows all the information about the project and also contains dashboard images . Second page is the analytical part which contains all the queries with its output and visualization .

🚖 Ola Ride Data Analysis Project
📌 Project Overview

This project focuses on analyzing ride data from Ola Cabs
to understand customer behavior, ride patterns, and 
operational efficiency. The goal is to extract meaningful 
insights that can help improve business performance and user
experience.

🎯 Objectives
Analyze ride booking trends
Identify peak hours and high-demand locations
Study ride cancellations and completion rates
Evaluate revenue and fare distribution
Provide data-driven business recommendations

📊 Dataset Description
The dataset includes:
Ride details (Ride ID, Date, Time)
Pickup & Drop locations
Fare amount & payment type
Ride status (Completed / Cancelled)
Customer & driver-related data
🛠️ Tools & Technologies Used
SQL (MySQL) – Data cleaning & querying
Power BI – Dashboard creation & visualization
Excel – Data preprocessing
DAX – KPI calculations

📈 Key KPIs
Total Rides
Total Revenue
Average Ride Fare
Cancellation Rate
Completion Rate
Peak Booking Time

📊 Dashboard Features
Ride volume trends over time
Location-based demand analysis
Revenue breakdown
Cancellation analysis
Interactive filters (Date, Location, Ride Type)

🔍 Key Insights
Ride demand is highest during peak office hours
Certain locations show consistently higher bookings
Cancellation rates impact overall revenue
Weekend trends differ from weekday patterns

💡 Recommendations
Increase driver availability during peak hours
Reduce cancellations through better driver allocation
Offer incentives in high-demand areas
Use dynamic pricing to optimize revenue
Improve customer experience with faster ride matching

🚀 How to Run the Project
Import dataset into MySQL
Perform data cleaning and analysis using SQL
Load data into Power BI
Build dashboards using KPIs
Use filters to explore insights

📌 Future Improvements
Add real-time ride tracking data
Implement predictive analytics for demand forecasting
Include weather and traffic data
Build a live dashboard
